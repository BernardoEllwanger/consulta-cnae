# Consulta CNAE

Ferramenta de busca para correlacionar códigos CNAE com itens da Lista de Serviços da Lei Complementar 116/2003 (ISS).

## Funcionalidades

- Busca por código CNAE, item da LC 116 ou descrição do serviço
- Filtragem por categoria de atividade
- Indica local de incidência do ISS e obrigatoriedade de retenção
- Autocomplete com navegação por teclado
- Paginação dos resultados
- Layout responsivo (desktop e mobile)
- Base com 210 registros — funciona 100% no navegador, sem backend

## Stack

HTML · CSS · JavaScript

## Como rodar

Abra `index.html` no navegador ou sirva os arquivos com qualquer servidor estático:

```bash
npx serve .
```
